      Remarks

      The ScriptParameter is a string that tells Tingen what work needs to be done,
      and how it should be done.

      The Script Parameter will generally contain the following components:

      ComponentDescription
      Module The Tingen Module that will do the work (e.g.: "Admin")
      Command The requested command (e.g.: "Service")
      Action The requested action (e.g.: "Update")
      Option The (optional) option (e.g.: "All")
      Some Script Parameters may not contain all of the above components, or may
      contain additional components that aren't listed.

      All Script Parameter components are separated by a "-" character.

      Script Parameters are pre-defined, so you can't just make up a new one and
      have it magically work. Sorry. You can find a list of currently supported
      Script Parameters here.

      Tingen converts Script Parameters to lowercase for easier parsing, so you can use
      whatever casing makes sense to you when creating a ScriptLink event.

      Example

      To update all service information, the Script Parameter would be:
      C#
      Admin-Service-Update-All
      And to verify that the current user is authorized to submit an open incident:
      C#
      OpenIncident-Verify-AuthorIsViewing
