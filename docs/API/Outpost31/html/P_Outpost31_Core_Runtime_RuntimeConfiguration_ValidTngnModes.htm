      Remarks

      The Tingen Web Service can have one of the following modes:

      • DISABLED – The Tingen Web Service does not do any work.
      • ENABLED – The Tingen Web Service functions normally.
      • PASSTHROUGH – The Tingen Web Service writes log files, but does not make any modifications to data.
