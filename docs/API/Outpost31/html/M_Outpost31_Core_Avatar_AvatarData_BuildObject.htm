      Remarks

      Before returning an OptionObject to Avatar, it must be finalized.

      Finalizing an OptionObject is a three part process:

      1. Assign an error code to the returnOptionObject
      2. Assign an error message to the returnOptionObject
      3. Verify the returnOptionObject contains all required data
      Valid error codes include:
      Error codeDescription
      0Success - Returns the OptionObject without a message
      1Error - Stops the script from processing and displays an message with an "OK" button
      2OKCancel - Displays a message with "OK" and "Cancel" buttons.
      3Info - Displays an informational warning message with an "OK" button.
      4YesNo - Displays a message with "Yes" and "No" buttons.
      5OpenURL - Opens an URL in a browser
      6OpenForm - Presents an OK/Cancel dialog to open a form (only be used on Form Load and Field events).
