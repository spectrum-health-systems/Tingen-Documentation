      Remarks

      1-----------------------------------------------------------------------------80-----|-------------------------|-----120

      Before returning an OptionObject data structure to Avatar, it must be
      formatted properly, and contain all required information. The Finalize
      process takes care of those things by:

      1. Cloning the ReturnOptionObject the WorkOptionObject
      2. Assigning a valid error code to the returnOptionObject
      3. Assigning an error message to the returnOptionObject
      4. Verifying the returnOptionObject contains all required data
      The OptionObject expects an error code of type int, but to make it
      more readable, Tingen works with the following "error strings" (which will
      then be converted to an int):
      Error stringDescription
      "clone",
      "none",
      "success"      		Returns the OptionObject without a message (error code 0)
      "error"Stops the script from processing and displays an message with an "OK" button (error code "1")
      "okcancel"Displays a message with "OK" and "Cancel" buttons (error code "2")
      "info"Displays an informational warning message with an "OK" button (error code "3")
      "yesno"Displays a message with "Yes" and "No" buttons (error code "4")
      "openurl"Opens an URL in a browser (error code "5")
      "openform"Presents an OK/Cancel dialog to open a form (error code "6")
      NOTE: The openform error string can only be used on Form Load and Field events.

      Example

      To stop a script from processing and display an error message with an "OK" button:
      C#
      Outpost31.Core.Avatar.ReturnObject.Finalize(tnSession, "error", "There was an error!");
      To pause a script and ask if the user wants to continue:
      C#
      Outpost31.Core.Avatar.ReturnObject.Finalize(tnSession, "yesno", "You might want to check your data. Do you want to continue?");
      To display an informational message:
      C#
      Outpost31.Core.Avatar.ReturnObject.Finalize(tnSession, "info", "Have a nice day!");
