      Remarks

      All of the imortant data sent from Avatar is stored here, and should only contain the
      following Avatar-specific data:

      • The SystemCode
      • The SentScriptParameter
      • The SentOptionObject
      • The WorkOptionObject
      • The ReturnOptionObject

      When Tingen starts, this object is created and added to the TingenSession object.
